## Run Locally

Clone the project

```bash
  git clone https://github.com/Nitesh-Samaniya/lens.git
```

Go to the project directory

```bash
  cd lens-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
